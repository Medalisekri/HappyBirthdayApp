* Description de l'application :

 * Application Happy Birthday :

Cette application Android simple est développée avec Jetpack Compose.
Elle affiche un message d’anniversaire au centre de l’écran, superposé à une image de fond avec un effet de transparence, offrant une interface moderne et élégante.

  *  Fonctionnalités

Interface moderne avec Jetpack Compose

Image de fond avec effet de transparence

Message d’anniversaire centré avec signature

Application légère et adaptée aux débutants

   * Technologies utilisées

Kotlin

Jetpack Compose

Android Studio
